
# 🚀 Portfolio Website Deployment on AWS using Terraform & GitHub CI/CD

Today I completed an exciting DevOps project where I successfully hosted my personal portfolio website on **AWS Cloud** using **Terraform** and automated deployment through **GitHub Pipeline (CI/CD)**.

## 📌 Project Overview

In this project, I used **Infrastructure as Code (IaC)** with Terraform to create and manage AWS resources, and connected my GitHub repository with an automated CI/CD pipeline for seamless deployment.

## 🛠️ Technologies Used

- **Terraform**
- **AWS Cloud**
- **GitHub Actions**
- **CI/CD Pipeline**
- **S3 / EC2 (Based on setup)**
- **HTML, CSS, JavaScript**
